Hafcons

Welcome!

Hafcons is a library of ready to use icons from Font Awesome with a simple hover effect using that comapny's / brand's main color. The 'haf' in Hafcons stands for hover, active, and focus - the 3 pseudo classes you should include for a hover effect on a link.

Usage

Make sure to add the Font Awesome CDN into the <head> section of your HTML document. Just click the button below to copy it:

CDN

Each icon card comes with a preview of what the icon would look like with a simple hover effect. Each icon card also contains a link to their respective site so you can learn more about them. Try out this React card for example:
